Program to calculate the movements and properties of the celestial bodies in our solar system.

When program is run number of years and numer of stepr pr year will be asked for aswell as the solver you would like.

Solutions and variables are dumped into two files in build-folder for later use.

Three python script is also available called "plot.py", "plot1.py" and "plot_mercurt" which can read the the two dumped files and plot the solutions in 3D and 2D.
